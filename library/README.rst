Installing
==========

Stable library from PyPi:

-  Just run ``sudo pip install envirophatplus``

Latest/development library from GitHub:

-  ``git clone https://github.com/pimoroni/envirophatplus-python``
-  ``cd envirophatplus-python``
-  ``sudo ./install.sh``
